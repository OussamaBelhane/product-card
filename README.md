# Product Card App

A React Native application demonstrating a "Product Card" component and a "Product List" using `FlatList`. This project was created as part of "TP SÃ©ance 7".

## Features

-   **Product Card**: A reusable component displaying product image, title, price, description, and a "Buy" button.
-   **Product List**: A scrollable list of products using `FlatList`.
-   **Total Price Panel**: A sticky panel at the top that updates the total price when products are "bought".
-   **Responsive Design**: Uses Flexbox for layout, avoiding absolute positioning.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/OussamaBelhane/product-card.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd product-card
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

## Usage

Start the Expo development server:

```bash
npx expo start
```

-   **Mobile**: Scan the QR code with the Expo Go app.
-   **Tunnel Mode**: If you have connection issues, use `npx expo start --tunnel`.
